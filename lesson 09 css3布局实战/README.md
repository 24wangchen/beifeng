	- 拿到图依然要有大局观
  - 百分比布局
  - 拿到2倍图，写代码要写一倍图
	- 找到要剔除的地方，切出整体

###特别的注意点
	- media query合理使用
	- 图片使用2倍以上的图
	- 1px的线需要用特别的方法生成
```css
.after-scale{
   position: relative;
}
.after-scale:after{
   content:"";
   position: absolute;
   bottom:0px;
   left:0px;
   right:0px;
   border-bottom:1px solid #c8c7cc;
   -webkit-transform:scaleY(.5);
   -webkit-transform-origin:0 0;
}
```
    
    -box-shadow的妙用

```css
.box-shadow{
    box-shadow:inset 0px -1px 1px  -1px #c8c7cc;
}
```
